### 构造函数的调用
```
class A{
	A(){};
};
```
①`A a`②`A arr[3]`③`A *p=new A [3]`都会调用构造函数，但是`A *n`不会调用构造函数。

***
以下代码段：
```
class A{
	A(){};
	A(int a){};
	A(A &a){};
};
int mian()
{
	A arr1[3]={1,2}; 
	//arr1[0]和arr1[1]调用第二个构造函数；arr1[2]调用无参构造函数
	A *arr2=new A[3]; 
	//三项均调用无参构造函数，因为已经动态分配了空间
	A *arr3[3]={new A(),NULL,new A()}; 
	//只是创立三个空指针，所以只有arr[0]和arr[2]调用两次构造函数
}
```
